Adding/Staging commands

  • add/commit for staging
  • reset HEAD <filename> - reverts staged changes latest head (undeletes, unchanges)
  • git checkout <filename> - restores most recently committed state
  • git rm - deletes from os and stages delete
  • git checkout (commit hash code) - go back in time to specified state
  • git checkout master(or name of other branch) - return to master branch/desired branch
  • git branch <new name> <hash code of latest commit (if alternate/previously discarded branch)>
  • git branch <new name> (with no hash code) clones current branch/master
  • git merge <branch to merge to master> - brings changes from branch to merge into master
  • git branch -m <oldname> <newname> - rename branch
  • git branch -D <name> - deletes branch <name>
  • cloned projects; git checkout -b <branch name> <location of branch on github (origin) origin/<name>> - check out this branch, switch to project branch
  • clone all branches: git clone -- mirror (link) .git (have to make directory for it first) git config --bool core.bare false <-- switch from bare to reg repo after cloning git reset --hard <-- resets HEAD/everything into normal repository

Stash

  • git stash save "change message" - saves changes that have not been committed yet separately.
  • above command also runs git reset --hard (above, takes latest current repo and puts into pwd)
  • --untracked option to also include untracked files
  • git stash list - shows all stashed states
  • git stash show <stash name from list> - can use -p option to show differences between head
  • git stash pop <name>/ git stash apply <name>- pop removes after taking out, apply leaves in stash
  • MAKE SURE IN CORRECT BRANCH BEFORE APPLYING
  • remove from stash? (directory state) - git stash drop <stash name>
  • mass delete - git stash clear

Pushing and stuff

  • origin/master is local copy of repo branch that reflects remote repo. update by fetching
  • push to update repo with local changes, merge to get remote changes
  • To add: git remote add origin [link]
  • pull: git pull [link] [which branch]
    • to pull only from specific branc: git push -u origin [branch name]
    • to push: git push -u origin master
    • Delete remote branch: git push origin :[branch name] or git push origin --delete [branch name]
